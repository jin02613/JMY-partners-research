Features

주요 기능

DART API 기반의 종합 재무분석 도구. 기업분석부터 전종목 스크리닝까지.

📈

기업개요

실시간 주가, 시가총액, PER·PBR·ROE 등 투자지표와 매출 구성비를 한 화면에 표시합니다. 네이버 금융에서 실시간 데이터를 크롤링합니다.

실시간 주가 투자지표 매출구성 52주 고/저
📊

실적 그래프 (10+ 차트)

매출·수익성, 현금흐름, 운전자본, 성장률, 가치평가 등 5개 카테고리 10개 이상의 Chart.js 차트를 제공합니다.

연환산/연간/분기 연결/개별 Chart.js
📋

10년 재무제표

손익계산서, 재무상태표, 현금흐름표를 10년치 테이블로 조회. 대항목/소항목 확장/축소, 연환산·연간·분기 전환을 지원합니다.

손익계산서 재무상태표 현금흐름표
💰

매출이익지수 & 주가

주가와 매출·영업이익·순이익 지수를 기준값 100으로 정규화하여 비교. 이익 추세와 주가 괴리를 시각적으로 파악합니다.

주가 오버레이 지수 정규화 이중축 차트
🔎

순운전자본 스크리닝

전종목 대상으로 수정 순운전자본(Adjusted NWC) 기준 스크리닝을 실행합니다. 실시간 진행률, 카테고리별 그룹핑, 엑셀 다운로드를 지원합니다.

전종목 스캔 실시간 진행률 Excel 다운로드
🛒

종목쇼핑

전체 상장사의 재무 지표를 일괄 수집하여 비교합니다. 캐시된 데이터를 우선 활용하고 미수집 종목만 네이버에서 가져옵니다.

전종목 수집 캐시 활용 Excel 다운로드
Tech Stack

기술 구성

Python Flask 기반 풀스택 웹 애플리케이션.

Backend

  • Flask (Python Web Framework)
  • OpenDartReader (DART API)
  • Pandas (데이터 처리)
  • ThreadPoolExecutor (병렬 처리)
  • BeautifulSoup (웹 크롤링)

Frontend

  • Chart.js 4.4.7 (차트 라이브러리)
  • Vanilla JavaScript (SPA)
  • CSS3 (네이버 스타일 UI)
  • Responsive Design

Data Sources

  • DART Open API (재무제표, 배당)
  • Naver Finance (주가, 지표)
  • KOSPI / KOSDAQ 상장사
  • File + Memory Cache
API

API 엔드포인트

15개 이상의 RESTful API로 모든 데이터를 제공합니다.

Method Endpoint 설명
GET /api/search 종목명 자동완성 검색
GET /api/overview 기업개요 (PER, PBR, ROE, 시가총액 등)
GET /api/data 분기별 재무데이터 (손익/재무상태/현금흐름)
GET /api/finstate 10년 재무제표 전체 구조
GET /api/realtime_price 실시간 주가 (네이버 크롤링)
GET /api/price_chart 10년 일별 주가 히스토리
GET /api/dividend 배당 데이터 (DPS, 배당성향, 수익률)
POST /api/nwc/screening 순운전자본 전종목 스크리닝 시작
GET /api/nwc/screening/status 스크리닝 진행 상태 조회
GET /api/nwc/download NWC 결과 Excel 다운로드
POST /api/shopping/collect 종목쇼핑 전종목 수집 시작
GET /api/shopping/results 종목쇼핑 결과 데이터 조회
